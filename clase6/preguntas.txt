¿Quién inventó el sistema de control de versión Git y por qué?

Se inventó para dar eficencia y confibialidad en el mantenimiento de aplicaciones con gran número de archivos con código. Su propósito es mantener registro de los cambios que se hacen sobre los archivos. En otras palabras, para mantener un historial de versiones de un mismo archivo.

¿A quién pertenece actualmente Github y por qué?

A Microsoft, para acercarse más al mundo del software libre y abierto. 

¿Hay otra forma que no sea la terminal para trabajar con GitHub?

A través de la aplicación web de GitHub.
